## 2.0 Libraries Overview

### 2.1 Domain Modules

*   **[Application Logic (app)](app/README.md)**: Contains the core business logic of the application.
*   **[User Interface (ui)](ui/README.md)**: Contains React components, layout definitions, and UI utilities.
*   **[SDK (sdk)](sdk/README.md)**: Contains Software Development Kits for external service integrations.
*   **[Utilities (util)](util/README.md)**: Contains shared utility functions used across multiple libraries.
*   **[Identity (identity)](identity/README.md)**: Authentication and identity management components.
*   **[Plugins (plugin)](plugin/README.md)**: Plugin system interfaces and implementations.
*   **[Scripts (script)](script/README.md)**: Shared scripts and automation tools.
*   **[Enterprise (enterprise)](enterprise/README.md)**: Enterprise-specific features and modules.
