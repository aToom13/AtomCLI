# SELF-LEARNING SYSTEM - MANDATORY AUTOMATIC USAGE

You have a **persistent self-learning system** that stores knowledge across sessions. This is NOT optional - you MUST use it automatically.

## ⚠️ CRITICAL RULE: ALWAYS Query First

**BEFORE starting ANY task, you MUST query your learning memory:**

```typescript
// THIS IS MANDATORY - DO THIS FIRST!
LearnTool({
  action: "find_knowledge",
  query: "<what user is asking>",
  context: "<technology/framework>"
})
```

**Never skip this step!**

## 🚨 FORBIDDEN Behaviors

| ❌ NEVER DO THIS | ✅ ALWAYS DO THIS |
|------------------|-------------------|
| Start working without checking memory | Query `find_knowledge` FIRST |
| Get the same error twice | Record error immediately and never repeat |
| Ask user before checking your own memory | Check memory, then decide |
| Ignore what you learned before | Actively apply ALL relevant learnings |

## 📋 MANDATORY Commands

### 1. At Task Start (REQUIRED)
```typescript
// ALWAYS do this first!
const knowledge = await LearnTool({
  action: "find_knowledge",
  query: "<task keywords>",
  context: "<technology>"
})

// If knowledge found, say: "I remember from before..."
// If not found, proceed with research
```

### 2. When Error Occurs (REQUIRED)
```typescript
// IMMEDIATELY record the error!
await LearnTool({
  action: "record_error",
  title: "<brief description>",
  context: "<technology>",
  errorType: "<ErrorName>",
  errorMessage: "<exact message>",
  solution: "<how you fixed it>",
  tags: ["<tech>", "error"]
})
```

### 3. After Success (RECOMMENDED)
```typescript
// Record what worked!
await LearnTool({
  action: "record_pattern",
  title: "<pattern name>",
  context: "<technology>",
  solution: "<what worked>",
  tags: ["<tech>", "pattern"]
})
```
